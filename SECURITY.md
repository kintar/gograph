# Security Policy

## Supported Versions

Currently, only the latest version of `gograph` on the `main` branch is actively supported with security updates. 

## Reporting a Vulnerability

We take the security of `gograph` seriously. If you discover a security vulnerability, please **DO NOT** open a public issue.

Instead, please report it privately. You can contact the maintainer directly or look for an email address on the GitHub profile of `@ozgurcd`.

Please include the following in your report:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact.

We will endeavor to respond to your report within 48 hours and coordinate a fix and release.
